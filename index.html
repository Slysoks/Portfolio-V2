Un site web pour ma classe

Ce site web étrait produit en PHP ainsi qu'avec du HTML pure.

Dans le cadre d'un projet personnel, j'ai créé un site web pour ma classe de 3ème. Aujourd'hui, le site n'est malheureusement plus en ligne, mais il était possible de discuter dans un chat en ligne, bien évidement à condition d'être connecté.

Ce site contenait une partie utilisateur et une partie administrateur. La partie utilisateur permettait de discuter dans un chat en ligne, de voir les photos et l'emplois du temps de la classe. La partie administrateur, elle, permettait de gérer les utilisateurs ainsi que les messages du chat.

À l'époque, je ne connaissais pas encore le hashage de mot de passe, donc les mots de passe étaient stockés en clair dans la base de données, ce que je ne ferais plus aujourd'hui.

Ce site m'a appris pas mal de choses, notamment le PHP, la gestion d'une base de données MySQL, mais aussi la gestion d'un serveur Apache. C'est notamment après ce projet que j'ai découvert le hashage de mot de passe.

Un site web pour commander des galettes

Ce site web était produit en PHP, intéragissant avec une base de données MySQL.

Dans le cadre d'un projet personnel, j'ai décidé de créer un site web pour permettre de commander des galettes de façon à passer et prendre en compte les commandes plus simplement et plus rapidement.

Il y a donc une page pour les “dégustateurs” (pour commander), ainsi qu'une page pour le “cuisinier” (pour voir les commandes et les préparer). Depuis l'interface des dégustateurs, il est possible de mettre à jour sa commande en cas d'erreur.

Je l'ai aujourd'hui passé sous Docker pour permettre une installation plus simple et plus rapide.

Ce projet m'as permis d'apprendre les bases du PHP, de la gestion d'une base de données MySQL, l'installation d'un serveur Apache, mais surtout, la base de Docker.

SpaceY

Ce site web était produit en HTML pure, CSS et JavaScript.

Dans le cadre d'un projet scolaire, j'ai décidé de créer un site web sur un sujet scientifique au choix. J'ai bien entendu parlé de ma passion, l'espace.

Ce site web était composé d'une seule page avec deux sections. Il y avait une section sur les planètes telluriques et une autre sur les planètes gazeuses.

J'ai intégré un système de vue 3D des planètes (avec Model-Viewer). Chaque section comportait le nom de la planète, une représentation 3D ainsi qu'un texte explicatif.

La partie de la description des planètes pouvait être cachée ou affichée par un simple clic sur la boîte de la planète. Celui-ci par contre est toujours disponnible en cliquant ici.

Ce projet m'a permis de découvrir le Model-Viewer de Google, et donc l'intégration de modules JavaScript dans une page web. Pour ce site web et sa présentation, j'ai obtenu la note de 20/20.
